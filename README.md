# tasker_max_bot
Таскер для мессенджера макс

# Основная кноцепция
Таскер бот добавляется в рабочий чат команды (например, стартап, хакатон, заказная разработка и т.д.).
1. Все пользователи могут создать задачу: добавить название (строка), описание (абзац), исполнителя (участника чата), дэдлайн задачи, дату оповещения (во время, за час, за день, за 2 дня). 
2. В общий чат отправляется напоминание в назначенное время об указанной задаче. 
3. Все пользователи чата могут просматривать список актуальных задач.
4. Пользователи могут отмечать выполненные задачи флажком.
5. Выполненные задачи удаляются из списка активных задач и помещаются в архив.
6. Архив доступен из меню приложения для просмотра выполненных задач. Отсортирован по дате с убыванием.

# Технические моменты
1. Используем TypeScript библиотеку для общения с максом.
2. Для хранения записей задач используем Postgres
3. Фронт делаем на библиотеке MAX_UI

# Запуск
`docker compose up` запустит бд и по адресу localhost:5432. Креды указаны в compose файле. Схема в db/init_schema/
